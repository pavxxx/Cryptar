🎓 AI Concepts Behind Cryptarithmetic

Understand the AI techniques that power the solver.

🧩

Constraint Satisfaction Problem (CSP)

A CSP is defined by:

  • Variables: Each unique letter in the puzzle (e.g., S, E, N, D, M, O, R, Y)
  • Domains: Possible values each variable can take (digits 0–9)
  • Constraints: The arithmetic equation must hold, each letter maps to a unique digit, leading letters ≠ 0
Example: In SEND + MORE = MONEY, we have 8 variables, each with domain {0-9}, and the constraint that 1000S + 100E + 10N + D + 1000M + 100O + 10R + E = 10000M + 1000O + 100N + 10E + Y
🔄

Backtracking Search

A depth-first search algorithm that:

  • Assigns values to one variable at a time
  • When a dead end is reached (no valid value), it backtracks to the previous variable
  • Systematically explores the search tree until a solution is found or all possibilities are exhausted
Time Complexity: O(d^n) where d = domain size (10) and n = number of variables. Worst case for 8 variables: 10^8 = 100,000,000 combinations!
✂️

Pruning & Constraint Propagation

Pruning reduces the search space by eliminating branches that cannot lead to a solution:

  • Column-wise validation: Check arithmetic constraints column by column as digits are assigned
  • Forward checking: If assigning a value makes any future variable unsatisfiable, prune immediately
  • Unique digit constraint: Once a digit is assigned, remove it from all other variables' domains
Impact: Pruning typically reduces the search space by 90–99%, turning minutes of computation into milliseconds!
🤖

Systematic Search Agent

The AI solver acts as an intelligent agent with:

  • State: The current partial assignment of letters to digits
  • Actions: Assign a digit to the next unassigned letter, or backtrack
  • Goal: A complete, valid assignment satisfying all constraints
  • Strategy: Depth-first search with constraint checking at each step (systematic search)
This solver uses a right-to-left column ordering heuristic — assigning letters from the units column first allows earlier constraint detection.
